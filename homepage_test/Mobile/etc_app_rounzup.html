놀랍도록 강력한
궁극의 기능으로요.

UP 1.

현실처럼 생생하게
실시간 가상피팅

어디서든 자유롭게.
라운즈는 Face alignment 기술과 증강현실(AR) 기술을 접목하여 세계 최고 수준의 가상피팅 서비스를 제공합니다.
구분이 어려울 정도로 생생하게 구현된 가상피팅은 원하는 상품을 시간과 장소 제약없이 마음껏 착용할 수 있도록 지원하죠.

UP 2.

궁극의 측정
정확한 내 얼굴형 분석

주변에 물어보지 않아도.
라운즈 A.I 딥러닝 분석 로직은 얼굴을 67개의 좌표로 구조화 하고 그 값을 이용해 이목구비의 비율을 계산하여 8가지의 대표 얼굴형으로 분류합니다. 게다가 A.I. 딥러닝은 다양한 얼굴의 좌표 데이터를 지속적으로 학습하여 분석의 정확도를 높입니다. 이제 내 얼굴을 가장 객관적인 데이터로 빠르게 파악할 수 있어요.

UP 3.

취향을 저격하는
정교한 상품 맞춤 추천

뛰어나고 탁월한 이해력으로.
아이웨어 쇼핑 데이터를 바탕으로 경향성과 선호를 파악하는 개인화 추천 모듈을 구축하고 있습니다. 검색어부터, 관심상품, 스타일까지 사용자의 관심사 하나 하나를 세심하게 파악하여 취향에 맞는 상품을 제안하죠.

UP 4.

손쉽게 그리고 빠르게
초간편 상품 탐색

뭉치면 쉬워진다.
원하는 상품을 쉽고 빠르게 찾을 수 있도록 탐색 기능을 통합했습니다. 브랜드, 안경테, 프레임, 인기상품 등을 탐색 조건을 쉽게 분류하고 선택할 수 있도록 한곳으로 모아 탐색 편의성을 높였습니다. 이제, 이곳 저곳에서 찾을 필요 없어요. 통합된 쇼핑 탐색도구로 한번에 찾아보세요.
